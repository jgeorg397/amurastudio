# Mazu Partners Website

A bold, modern, mobile-first website for Mazu Partners, a founder-led investment firm. The design is inspired by the aesthetic of a11.global with a dark background, blue accents, and confident typography.

## Features

- **Responsive Design**: Mobile-first approach that works on all screen sizes
- **Modern Aesthetics**: Dark theme with bold blue (#0066FF) accents
- **Smooth Animations**: Scroll-triggered fade-ins and subtle motion effects
- **Clean Typography**: Inter font family with confident, large text
- **Interactive Elements**: Hover effects, form validation, and smooth scrolling
- **No Dependencies**: Pure HTML, CSS, and vanilla JavaScript

## Sections

1. **Hero Section**: Bold headline with animated background elements
2. **Our Focus**: Two investment focus areas with clear messaging
3. **What Makes Us Different**: Four key differentiators with custom icons
4. **About/Credibility**: Founder background and experience highlights
5. **Contact/CTA**: Contact form and email link

## Color Palette

- **Background**: #000000 (pure black)
- **Secondary Background**: #0A0A0A (near-black)
- **Accent**: #0066FF (bold blue)
- **Text**: #FFFFFF (headlines), #CCCCCC (body copy)
- **Muted Text**: #999999 (secondary text)

## Typography

- **Font Family**: Inter (Google Fonts)
- **Headlines**: 700-800 weight, large sizes with negative letter-spacing
- **Body Text**: 400 weight, optimized for readability

## Animations

- Scroll-triggered fade-in animations
- Subtle parallax effects on hero background
- Hover states with transforms and shadows
- Animated blue lines and shapes in hero section

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)
- Progressive enhancement approach

## File Structure

```
├── index.html          # Main HTML structure
├── styles.css          # All styling and responsive design
├── script.js           # JavaScript for animations and interactions
└── README.md           # This documentation
```

## Getting Started

1. Open `index.html` in a web browser
2. No build process or dependencies required
3. All assets are self-contained

## Customization

- Colors can be modified in the CSS variables section
- Typography can be adjusted by changing font weights and sizes
- Animations can be disabled by removing the `fade-in` classes
- Form submission can be connected to a backend service

## Performance

- Optimized for fast loading
- Minimal JavaScript footprint
- Efficient CSS with modern features
- No external dependencies or heavy libraries
